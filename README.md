# Manifests

Collection of manifests to work with [repo](https://gerrit.googlesource.com/git-repo/)

## Workspace Init

`repo` will be use to checkout the full workspace:

```sh
$ repo init -u ssh://git@github.com/Mord3rca/manifests
$ repo sync
```

Once initialized and synced, see [repo-workspace](https://github.com/Mord3rca/repo-workspace) for product compilation
